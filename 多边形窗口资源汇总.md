## 🚀 5种实现方案速查

### 方案1: CSS clip-path (推荐⭐⭐⭐⭐⭐)
```css
.trapezoid {
  clip-path: polygon(10% 0%, 90% 0%, 95% 100%, 5% 100%);
  border: 2px solid rgba(0, 180, 255, 0.7);
  box-shadow: 0 0 20px rgba(0, 180, 255, 0.5);
}
```
✅ 性能最优 | ✅ 兼容性好 | ✅ 代码简单

### 方案2: SVG Polygon
```html
<svg viewBox="0 0 1000 600">
  <polygon points="100,0 900,0 1000,600 0,600"
           stroke="#00dcff" stroke-width="3" fill="none"/>
</svg>
```
✅ 灵活度高 | ✅ 动画丰富 | ⚠️ 性能一般

### 方案3: Border + Transform
```css
.trapezoid {
  border-bottom: 100px solid blue;
  border-left: 25px solid transparent;
  border-right: 25px solid transparent;
}
```
✅ 兼容IE | ⚠️ 灵活性低 | ⚠️ 难以添加边框

### 方案4: 伪元素遮盖
```css
.box::before {
  content: '';
  background: #000;
  /* 遮盖不需要的边框部分 */
}
```
✅ 兼容性好 | ⚠️ 代码复杂 | ⚠️ 难维护

### 方案5: augmented-ui 库
```html
<div data-augmented-ui="tl-clip tr-clip border">
  Content
</div>
```
✅ 使用简单 | ✅ 效果丰富 | ⚠️ 需要引入库

## 💡 最佳实践建议

### 性能优先
1. ✅ 首选 CSS clip-path
2. ✅ 动画用 transform 和 opacity
3. ⚠️ 谨慎使用 SVG filter (移动端)

### 兼容性考虑
```css
/* 提供降级方案 */
.trapezoid {
  border: 2px solid #00dcff; /* 不支持clip-path时的回退 */
  clip-path: polygon(...);
}

@supports (clip-path: polygon(0 0)) {
  .trapezoid {
    /* 增强效果 */
  }
}
```

### 响应式设计
```css
.trapezoid {
  width: 80%;
  max-width: 800px;
  aspect-ratio: 3 / 1;
}

@media (max-width: 768px) {
  .trapezoid {
    clip-path: polygon(2% 0%, 98% 0%, 100% 100%, 0% 100%);
  }
}
```
