# 外网访问快速指南

## 🚀 30秒快速配置

假设您的新项目：
- 前端运行在 3000 端口
- 后端运行在 3001 端口
- 想通过 8888 端口对外访问

### 执行命令：

```bash
# 1. 生成配置
./setup_external_access.sh myproject 3000 3001 8888

# 2. 应用配置（需要sudo密码）
sudo cp /tmp/myproject-nginx.conf /etc/nginx/sites-available/myproject
sudo ln -sf /etc/nginx/sites-available/myproject /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

### 路由器设置：
- 外部端口：8888
- 内部IP：192.168.0.20
- 内部端口：8888
- 协议：TCP

### 访问地址：
```
http://67.169.169.65:8888
```

## 📋 已占用端口列表

| 端口 | 用途 |
|------|------|
| 80 | LFWLW项目nginx |
| 3389 | 远程桌面 |
| 50000 | LFWLW前端（外网） |
| 50001 | LFWLW后端 |

## ⚠️ 关键提醒

1. **不要在项目中硬编码API地址**
2. **使用相对路径调用API**
3. **开发环境CORS设置为 `*`**

## 🔧 故障排查

```bash
# 检查配置
./check_external_access.sh

# 查看日志
sudo tail -f /var/log/nginx/error.log
```
