# 八边形窗口更新说明

## ✅ 已完成的修改

### 1. CSS Clip-path 更新为八边形

**之前 (梯形 - 4个点):**
```css
clip-path: polygon(
  8% 0%,      /* 左上 */
  92% 0%,     /* 右上 */
  98% 100%,   /* 右下 */
  2% 100%     /* 左下 */
);
```

**现在 (八边形 - 8个点):**
```css
clip-path: polygon(
  15% 0%,     /* 顶部左 */
  85% 0%,     /* 顶部右 */
  100% 15%,   /* 右上 */
  100% 85%,   /* 右下 */
  85% 100%,   /* 底部右 */
  15% 100%,   /* 底部左 */
  0% 85%,     /* 左下 */
  0% 15%      /* 左上 */
);
```

### 2. SVG 边框更新为八边形

**之前 (梯形):**
```html
<polygon points="80,0 920,0 980,600 20,600" />
```

**现在 (八边形):**
```html
<polygon points="150,0 850,0 1000,150 1000,450 850,600 150,600 0,450 0,150" />
```

### 3. 角点装饰升级

**之前:** 4个角的L型装饰
**现在:** 8个角的圆形脉冲装饰

每个角点包含:
- 实心圆点 (r="6")
- 外圈光晕 (r="10", 半透明)
- 脉冲动画效果

### 4. 新增动画

```scss
@keyframes cornersPulse {
  0%, 100% {
    opacity: 0.6;
    filter: brightness(1);
  }
  50% {
    opacity: 1;
    filter: brightness(1.5);
  }
}
```

## 📐 八边形坐标说明

基于 1000x600 的 SVG viewBox:

```
        (150,0)────────────(850,0)
       /                          \
    (0,150)                    (1000,150)
      |                              |
      |                              |
    (0,450)                    (1000,450)
       \                          /
        (150,600)──────────(850,600)
```

转换为百分比 (用于 CSS clip-path):
- 顶部边: 15% ~ 85% (宽度70%)
- 底部边: 15% ~ 85% (宽度70%)
- 左侧边: 15% ~ 85% (高度70%)
- 右侧边: 15% ~ 85% (高度70%)

## 🔧 自定义调整

### 调整八边形"尖锐度"

```css
/* 更方正的八边形 (角度小) */
clip-path: polygon(
  20% 0%, 80% 0%,
  100% 20%, 100% 80%,
  80% 100%, 20% 100%,
  0% 80%, 0% 20%
);

/* 更尖锐的八边形 (角度大) */
clip-path: polygon(
  10% 0%, 90% 0%,
  100% 10%, 100% 90%,
  90% 100%, 10% 100%,
  0% 90%, 0% 10%
);
```

### 调整角点大小

```tsx
// 在 PolygonBorder.tsx 中
<circle cx="150" cy="0" r="8" />  // 更大的角点
<circle cx="150" cy="0" r="4" />  // 更小的角点
```

### 调整角点动画速度

```scss
animation: cornersPulse 1s ...;  // 更快
animation: cornersPulse 3s ...;  // 更慢
```
