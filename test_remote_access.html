🌐 LFWLW 远程访问配置测试

📋 实施步骤

步骤 1: 运行 ./install_nginx.sh 安装nginx
步骤 2: 运行 ./configure_nginx.sh 配置反向代理
步骤 3: 运行 ./check_nginx_status.sh 检查状态

⚠️ 前置要求

  • ✓ 前端服务运行在 http://localhost:50000
  • ✓ 后端服务运行在 http://localhost:50001
  • ✓ 防火墙开放80端口
  • ✓ 域名正确解析到服务器IP

🛠️ 故障排查命令

# 查看nginx状态
sudo systemctl status nginx

# 查看错误日志
sudo tail -f /var/log/nginx/error.log

# 测试配置文件
sudo nginx -t

# 重启nginx
sudo systemctl restart nginx
