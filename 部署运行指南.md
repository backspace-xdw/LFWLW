# 物联网远程监控平台 - 部署运行指南

## 一、快速启动步骤

### 1. 环境准备
确保系统已安装：
- Node.js >= 16.0.0
- npm >= 8.0.0

### 2. 启动后端服务
```bash
# 进入后端目录
cd /home/shenzheng/lfwlw/backend

# 安装依赖（首次运行）
npm install

# 创建环境配置文件
echo 'PORT=5000
NODE_ENV=development
JWT_SECRET=dev-secret-key-2024
JWT_EXPIRES_IN=7d
CORS_ORIGIN=http://localhost:3000' > .env

# 启动后端服务
npm run dev
```

### 3. 启动前端服务（新开终端）
```bash
# 进入前端目录
cd /home/shenzheng/lfwlw/frontend

# 安装依赖（首次运行）
npm install

# 启动前端服务
npm run dev
```

### 4. 访问系统
- 打开浏览器访问：http://localhost:3000
- 登录账号：
  - 用户名：`admin`
  - 密码：`admin123`

## 四、常见问题解决

### 问题1：npm install失败
```bash
# 清理缓存
npm cache clean --force
# 删除node_modules
rm -rf node_modules package-lock.json
# 重新安装
npm install
```

### 问题2：端口被占用
```bash
# 查看端口占用
lsof -i :3000  # 前端
lsof -i :5000  # 后端

# 结束占用进程
kill -9 <PID>
```

### 问题3：实时数据不更新
1. 检查WebSocket连接状态（仪表板右上角）
2. 刷新页面重新连接
3. 检查浏览器控制台错误

### 问题4：登录失败
1. 确认后端服务已启动
2. 检查用户名密码是否正确
3. 查看后端控制台日志

## 六、部署到生产环境

### 1. 后端部署
```bash
# 构建
cd backend
npm run build

# 使用PM2运行
npm install -g pm2
pm2 start dist/index.js --name iot-backend
```

### 2. 前端部署
```bash
# 构建
cd frontend
npm run build

# 部署到Nginx
cp -r dist/* /var/www/html/
```

### 3. 环境变量配置
生产环境需要修改：
- JWT_SECRET：使用强密码
- DATABASE_URL：连接实际数据库
- CORS_ORIGIN：配置实际域名
