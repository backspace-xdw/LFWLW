# ISAFV-QHSE AIoT Platform 项目进度

## 项目概述
- **项目名称**: ISAFV-QHSE AIoT Platform (原LFWLW物联网监控平台)
- **技术栈**: 
  - 前端: React + TypeScript + Vite + Ant Design
  - 后端: Node.js + Express + TypeScript + Socket.IO
  - 实时通信: WebSocket + Socket.IO
  - 地图: 高德地图 API
- **部署地址**: http://ljinvestment.diskstation.me:50000/
