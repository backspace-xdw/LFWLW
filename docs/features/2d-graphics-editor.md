# 2D图形编辑器使用指南

## 概述
2D图形编辑器是一个基于Konva.js的工业图形编辑工具，支持P&ID（管道和仪表图）标准符号，可用于创建工业流程图并实时显示设备数据。

## 功能特性

### 1. 工业符号库
支持以下P&ID标准符号：
- **泵（Pump）**: 离心泵符号，带旋转动画
- **阀门（Valve）**: 手动/自动阀门，显示开度
- **储罐（Tank）**: 立式储罐，带液位显示
- **传感器（Sensor）**: 温度/压力传感器
- **电机（Motor）**: 三相电机符号
- **管道（Pipe）**: 连接管道

### 2. 编辑操作
- **添加图形**: 从左侧符号库拖拽到画布
- **选择图形**: 单击图形进行选择
- **移动图形**: 拖拽已选中的图形
- **删除图形**: 选中后按Delete键或点击删除按钮
- **属性配置**: 选中图形后在右侧面板配置

### 3. 数据绑定
每个图形可以绑定到实际设备的实时数据：
1. 选中图形
2. 在属性面板选择设备ID
3. 图形将自动显示该设备的实时数据
4. 支持显示多个参数（温度、压力、流量等）

### 4. 场景管理
- **保存场景**: 点击工具栏的保存按钮
- **加载场景**: 从已保存的场景列表选择
- **导出图片**: 将当前场景导出为PNG图片

## 使用步骤

### 创建新场景
1. 进入"2D图形编辑"页面
2. 从左侧符号库拖拽所需符号到画布
3. 调整图形位置和大小
4. 配置图形属性和数据绑定
5. 保存场景

### 编辑现有场景
1. 点击"加载场景"按钮
2. 选择要编辑的场景
3. 进行修改
4. 保存更新

### 实时监控
1. 创建场景并绑定设备数据
2. 保存场景
3. 在监控模式下查看实时数据更新
4. 图形会根据数据值自动更新显示

## 技术实现

### 前端架构
```
GraphicsEditor/
├── index.tsx          # 主页面组件
├── Canvas.tsx         # Konva画布组件
├── Toolbar.tsx        # 工具栏
├── SymbolLibrary.tsx  # 符号库面板
├── PropertyPanel.tsx  # 属性配置面板
└── symbols/          # 工业符号组件
    ├── Pump.tsx
    ├── Valve.tsx
    ├── Tank.tsx
    └── ...
```

### 状态管理
使用Zustand + Immer管理编辑器状态：
- 图形列表
- 选中状态
- 场景配置
- 撤销/重做历史

### 数据流
1. Socket.IO接收实时设备数据
2. 根据图形的设备ID匹配数据
3. 更新图形显示属性
4. Konva自动重绘画布

## 扩展开发

### 添加新符号
1. 在`symbols/`目录创建新组件
2. 实现符号的绘制逻辑
3. 添加到符号库配置
4. 实现数据绑定接口

### 自定义属性
1. 扩展图形数据结构
2. 在PropertyPanel添加控件
3. 实现属性更新逻辑
4. 保存到场景配置

## 注意事项
- 大型场景建议分页或分组管理
- 实时数据更新频率可配置
- 导出图片会包含当前数据值
- 场景数据保存在本地存储
