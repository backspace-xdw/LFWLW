# 物联网远程监控管理Web平台 - 模块接口定义

## 1. 接口设计原则

### 1.1 设计规范
- **RESTful风格**: 资源导向，使用HTTP动词
- **统一响应格式**: 标准化的成功/错误响应结构
- **版本管理**: URL路径包含版本号 `/api/v1/`
- **认证方式**: Bearer Token (JWT)
- **分页规范**: 统一的分页参数和响应
- **错误码规范**: 业务错误码体系

### 1.2 通用响应格式

```typescript
// 成功响应
interface SuccessResponse<T> {
  code: 0;
  message: 'success';
  data: T;
  timestamp: number;
  requestId: string;
}

// 错误响应
interface ErrorResponse {
  code: number; // 错误码
  message: string; // 错误信息
  details?: any; // 详细错误
  timestamp: number;
  requestId: string;
}

// 分页响应
interface PageResponse<T> {
  code: 0;
  message: 'success';
  data: {
    items: T[];
    total: number;
    page: number;
    pageSize: number;
    totalPages: number;
  };
  timestamp: number;
  requestId: string;
}
```

### 1.3 错误码规范

```typescript
enum ErrorCode {
```

```graphql
  # 设备数据订阅
```

## 12. SDK示例

### 12.1 JavaScript/TypeScript SDK
```typescript
// SDK初始化
import { IotPlatformSDK } from '@iot-platform/sdk';

const sdk = new IotPlatformSDK({
  baseUrl: 'https://api.iot-platform.com',
  apiKey: 'your-api-key',
  version: 'v1'
});

// 设备操作

const device = await sdk.devices.get('PUMP_001');
await sdk.devices.control('PUMP_001', 'start', { speed: 50 });

// 数据查询
const realtimeData = await sdk.data.realtime('PUMP_001');
const history = await sdk.data.history('PUMP_001', {
  pointId: 'temperature',
  startTime: '2024-01-01T00:00:00Z',
  endTime: '2024-01-02T00:00:00Z'
});

// 实时订阅
sdk.subscribe.deviceData(['PUMP_001', 'VALVE_002'], (data) => {
  console.log('Received update:', data);
});

// 告警监听
sdk.subscribe.alarms({ severity: ['high', 'critical'] }, (alarm) => {
  console.log('New alarm:', alarm);
});
```
