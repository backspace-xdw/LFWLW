LFWLW 物联网平台 - 外网访问问题解决

⚠️ 问题原因

Vite 5.x 版本引入了严格的主机检查，默认拒绝非本地主机名的访问。

🔧 临时解决方案

  1. 使用本地端口转发（推荐） 
    ssh -L 50000:localhost:50000 -L 50001:localhost:50001 shenzheng@ljinvestment.diskstation.me
    然后访问: http://localhost:50000
  2. 使用 nginx 反向代理 
    # nginx 配置
server {
    listen 80;
    server_name ljinvestment.diskstation.me;
    
    location / {
        proxy_pass http://localhost:50000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
    
    location /api {
        proxy_pass http://localhost:50001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
  3. 构建生产版本 
    cd /home/shenzheng/lfwlw/frontend
npm run build
npx serve -s dist -l 50000 --cors

✅ 当前服务状态

  • 后端服务运行在: 50001 端口
  • 前端开发服务器运行在: 50000 端口（仅本地访问）

📱 开发建议

开发阶段，建议使用以下方式之一：

  1. 本地开发，完成后部署到服务器
  2. 使用 SSH 隧道进行远程开发
  3. 使用 VS Code Remote 或类似工具
  4. 配置 nginx 作为统一入口
