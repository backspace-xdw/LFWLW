# 竞品分析：研华WebAccess/SCADA

## 基本信息
- **公司**: 研华科技(Advantech)
- **产品**: WebAccess/SCADA
- **网站**: https://www.advantech.com/en-us/products/webaccess-scada
- **定位**: 100% Web-based SCADA软件解决方案/IIoT平台
- **分析时间**: 2025-06-28

## 1. 产品概述

### 1.1 核心定位
WebAccess/SCADA是研华推出的100%基于Web的SCADA软件解决方案和工业物联网(IIoT)平台，具有开放接口用于开发IoT应用程序。

### 1.2 主要功能
- **数据网关功能**: 从地面设备获取数据并通过MQTT发布/订阅传输到云应用
- **HTML5仪表板**: 支持跨平台、跨浏览器的数据分析
- **实时监控**: 100%基于Web的远程工程、监控和控制

### 2.2 协议支持广度
**工业标准协议**:
- Modbus RTU/TCP (工业控制标准)
- OPC UA/DA (工业自动化)
- Ethernet/IP (工业以太网)
- DNP3 (电力系统)
- BACnet (楼宇自动化)
- SNMP (网络管理)

**物联网协议**:
- MQTT (设备通信)
- SignalR (实时Web通信)
- RESTful API (系统集成)
